# KNIME for Chinese Studies

This is a collection of use cases for using KNIME in doing research in Chinese Studies. The documentation is in English and Chinese. This is the deliverable for the 2022 KNIME Teaching Award I received from KNIME AG. I greatly appreciate the support from KNIME AG, especially Aline Bessa, for her patience and support. 

## How to use this documentation

This documentation provides a collection of use cases for using KNIME in doing research in Chinese Studies. It would not cover every aspect of KNIME, as it is a very broad topic. You can find the complete documentation for KNIME [here](https://www.knime.com/).
