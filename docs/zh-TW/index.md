## 如何在 Windows 上安裝 KNIME

在 Windows 電腦上安裝 KNIME Analytics Platform，請按照以下步驟操作：

1. 前往 KNIME 網站的下載頁面。
2. 在下載頁面上，找到「Download KNIME」 (下載 KNIME) 的標籤。
3. 選擇符合您作業系統 (Windows) 的安裝選項。您可以在 Windows 安裝程式、自我解壓縮封存檔或 zip 壓縮檔之間選擇。
    - Windows 安裝程式 會解壓縮安裝資料夾、在您的桌面上新增圖示，並建議合適的記憶體設定。
    - 自我解壓縮封存檔 會建立一個包含 KNIME 安裝檔案的資料夾，而無需任何特殊軟體。
    - zip 壓縮檔 可讓您下載、儲存檔案並將其解壓縮到您選擇的位置。
4. 閱讀並接受隱私權政策和條款與條件，然後按一下「Download」(下載)。
5. 下載完成後，請繼續安裝：
    - 如果您下載的是安裝程式或自我解壓縮封存檔，請執行下載的檔案。
    -如果您下載的是 zip 壓縮檔，請將其解壓縮到您選擇的位置。然後，執行 knime.exe 以啟動 KNIME Analytics Platform。
6. KNIME Analytics Platform 將會啟動。系統可能會詢問您是否要與 KNIME 分享您的使用統計資料。不會分享任何個人資訊，這些統計資料將用於改善 Workflow Coach。
7.「Workspace Launcher」(工作區啟動器) 視窗將會出現，您需要在其中指定工作區的路徑。工作區是儲存您所有偏好設定和工作流程的資料夾。預設的工作區資料夾是 ..\knime-workspace，但您可以變更它。
